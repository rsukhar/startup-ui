# Обновление документации

## Локальный запуск документации для разработки

```sh
cd docs
npm run docs:dev -- --port 5555
```

## Обновляем документацию

```sh
cd docs
npm run docs:build
```

Заливаем содержимое `docs/.vitepress/dist/*` в корень хостинга, обслуживающего **startup-ui.ru** (по FTP/SFTP — реквизиты в настройках проекта).

> В сборку входит `.htaccess` (копируется из `docs/public/`). Он нужен Apache-хостингу, чтобы «чистые» URL без `.html` (например `/pages/components/forms/sform`) отдавали соответствующий `sform.html`, иначе будет 404.
>
> Это **скрытый файл** — многие FTP-клиенты по умолчанию его не показывают и не заливают. Включи показ скрытых файлов и убедись, что `.htaccess` оказался в корне рядом с `index.html`.

Коммитим и пушим изменения.
