# Знакомство

## Основы

### Что такое Startup UI

### Установка

```js
npm install startup-ui
```

Подключение стилей
```js
// app.js
import 'startup-ui/dist/index.css';
```

Подключение компоентов
```js
// Index.vue
import { SButton } from 'startup-ui'
```

### Обновление документации

```js
npm install --prefix ./packages/startup-ui
```

Устанавливаем зависимости vite-press

```js
cd docs

npm install
```

Запускаем локальный dev-сервер

```js
npm run docs:dev
```

Сборка

```js
npm run docs:build
```
